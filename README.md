# Manufacturing OEE Optimization Project

This project analyzes manufacturing performance using OEE methodology.

## Features
- OEE Calculation (Availability, Performance, Quality)
- Cost per Unit Analysis
- Cycle Time & Lead Time
- Pareto & Histogram Analysis
- Correlation Matrix
- Downtime Reduction Simulation

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python manufacturing_oee_analysis.py
